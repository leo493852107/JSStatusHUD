## JSStatusHUD

### How to use

#### 加载成功
```objective-c
[JSStatusBarHUD showSuccess:@"加载成功!"];
```

#### 加载失败
```objective-c
[JSStatusBarHUD showError:@"加载失败!"];
```

#### 正在加载
```objective-c
[JSStatusBarHUD showLoading:@"正在加载中..."];
```

#### 隐藏
```objective-c
[JSStatusBarHUD hide];
```

#### 普通文字
```objective-c
[JSStatusBarHUD showMessage:@"普通文字"];
```
